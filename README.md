# Conta-bancaria-Java
Projeto conta bancária curso POO Gustavo Guanabara feito em Java puro no bloco de notas


	ContaBancaria
atributos     metodos 
nº da conta   abrir conta()
tipo          fecharConta()
dono          depoistar()
saldo         sacar()
status        pagarMensal()

   requisitos
tipo(cc ou cp)
qnd abrir conta: status = true
se abrir cc: ganha $50
se for cp: ganha $150
para fechar conta o saldo deve ser 0.
para depositar conta precisa estar aberta
para sacar tem que tá aberta e tem q ter saldo para fazer o saque
mensalidade cc paga 12 reais  cp paga 20 reais
