# Conta-bancaria-Java
Projeto conta bancária curso POO Gustavo Guanabara feito em Java puro no bloco de notas
------------
ContaBancaria 
----------
atributos   
nº da conta  
tipo  
dono  
saldo  
status   

métodos  
abrir conta( )   
fecharConta( )  
depositar( )  
sacar( )   
pagarMensal( )  

requisitos  
tipo(cc ou cp)  
qnd abrir conta: status = true  
se abrir cc: ganha $50  
se for cp: ganha $150  
p/ fechar conta o saldo deve ser 0.  
p/ depositar conta precisa estar aberta  
p/ sacar tem que tá aberta e tem q ter saldo para fazer o saque  
mensalidade cc paga 12$  cp paga 20$  

Faça download do projeto compile através do CMD com os comandos:  
javac Teste.java  
java Teste
